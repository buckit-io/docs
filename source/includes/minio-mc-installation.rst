.. tab-set::

   .. tab-item:: macOS / Linux

      Install ``bm`` with the official Buckit Manager installer:

      .. code-block:: shell
         :class: copyable

         curl -fsSL https://buckit-io.github.io/bm/install.sh | sh
         bm --help

      The installer downloads the latest stable build for your operating
      system and architecture, verifies its SHA-256 checksum, and installs it
      into your user account.

   .. tab-item:: Windows PowerShell

      Install ``bm`` with the official Buckit Manager installer:

      .. code-block:: powershell
         :class: copyable

         irm https://buckit-io.github.io/bm/install.ps1 | iex
         bm --help

      The installer downloads the latest stable build for your operating
      system and architecture, verifies its SHA-256 checksum, and installs it
      into your user account.
